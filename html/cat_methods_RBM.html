Atlas creation and ROI based analysis (in development)

CAT12 allows estimation of tissue volume [, intensity, and thickness] for different atlas maps.

Results were stored in a csv-file catROI[SPACE]_[ATLAS]_*.csv for each atlas map ATLAS and in the cat_*.XML file. SPACE describes the space of the value estimation an can be s for subject space and w for normalized space. You can use external software like Excel for analysis or the cat_io_csv function to read the data as table, or the cat_io_xml.

We prepared different atlas maps for the CAT12 IXI555 space that are stored in the templates_1.50mm directory. For each atlas you can find a csv-file with the ROI IDs, short and full names. The txt-file contain further informations, terms of use, and the name of the original publications and websites.

To use the atlas maps with SPM atlas functions, you have to call the cat_install_atlases function once. By default only Neuromorphometics, LPBA40, and Hammers atlas maps were used.

List of atlas maps in CAT12

  • Neuromorphometics

    Maximum probability tissue labels derived from the "MICCAI 2012 Grand Challenge and Workshop on Multi-Atlas Labeling" (Challenge_Details). These data were released under the Creative Commons Attribution-NonCommercial (CC BY-NC) with no end date. Users should credit the MRI scans as originating from the OASIS project and the labeled data as "provided by Neuromorphometrics, Inc. (http://Neuromorphometrics.com/) under academic subscription" . These references should be included in all workshop and final publications.

    Display LPBA40 atlas.

  • LPBA40

    The LONI Probabilistic Brain Atlas (LPBA40) is a series of maps of brain anatomic regions [Shattuck:2008]. These maps were produced from a set of whole-head MRI of 40 human volunteers. Each MRI was manually delineated to identify a set of 56 structures in the brain, most of which are within the cortex. These delineations were then transformed into a common atlas space to produce a set of coregistered anatomical labels. The original MRI data were also transformed into the atlas space.

    Display LPBA40 atlas.

  • Hammers atlas:

    This atlas, based on Alexander Hammers' brain atlas based on 20 subjects, made available for the Euripides project, Nov 2009 [Hammers:2003,Gousias:2008].

    Display Hammers atlas.

  • IBSR

    18 subjectes of the IBSR V2 dataset, with low quality T1 data, but good labels maps. See IBSR terms in "IBSR.txt" for use.

    Display IBSR atlas.

  • Mori atlas

    This atlas based on the "Slicer3:Mori_Atlas_labels_JHU-MNI_SS_Type-II" atlas of Version 2010-05 [Oishi:].

    Display Mori atlas.

  • AAL (AUTOMATED ANATOMICAL LABELING) atlas:

    Combining probabilistic cytoarchitectonic maps and functional imaging data aligned on Collins brain [Collins:1998,Tzourio-Mazoyer:2002].

    Display AAL atlas.

Atlas creation with original data (Hammers, IBSR, Mori)

If original T1 maps and subject label maps were available we used CAT12 to preprocess the data and map them to common space. The subject dependent labels were translated to the CAT12 name system.
