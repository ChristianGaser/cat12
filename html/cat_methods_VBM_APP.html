Affine PreProcessing (APP) (in development):

APP was developed to process animal data with the standard CAT pipeline, because this data requires a good brain mask for SPM preprocessing. It include a simple inhomogeneity correction to improve the affine registration and to allow an initial skull-stripping that can be used as mask in the SPM preprocessing. APP can also help in standard human data, if the affine registration got problems by strong inhomogeneities. Using of brain masked data also reduce calculation times of the SPM preprocessing and it further helps in case of synthetic images. There are four options:

  • APP0: no APP and only standard preprocessing (default)
  • APP1: APP with simple bias correction
  • APP2: APP with improved bias correction
  • APP3: APP with improved bias correction, brain mask, but without initial affine registration for processing of other species (AC-PC has to be correct!)
