CAT defaults and user modes

CAT uses different user modes to avoid overpacking of the GUI by unimportant parameters that are irrelevant for most users. You can switch to default/expert mode by the commands cat12('default')/cat12('expert').

All CAT default parameter are stored in the cat_defaults file, that allows permanent changes of specific parameters. Furthermore, it is possible to use your own default files, by creating a copy of the cat_defaults file, change its parameters and load it from the command line by cat12('select'), cat12('cat_defaults_mydefs') or cat12('mypath/cat_defaults_mydefs').
