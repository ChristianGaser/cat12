CAT > Quick start guide

CAT test:

SPM delivers a 2 mm version of the Collins brain that can be processed by the cat_tst_cattest batch script that creates the cattest/CAT#_R# subdirectory and runs an example preprocessing that allows basic testing of the CAT main preprocessing functions. The subscripts of the different processing steps are located in the batches/cattest subdirectory and can be used as example for batch processing. The include VBM as well as SBM preprocessing and further functions to generate, extract, import, export, and combine further VBM, SBM, and VBM measures. Please keep in mind that the data has only 2 mm resolution that is acceptable for basic tests.

This script is still in development!

Subscipts:

  • cat12_SBM_101_segment, cat12_SBM_103_segment_greaterapes, cat12_SBM_103_segment_oldworldmonkeys

    Preprocessing scripts of the menu entry Segment Data that create (a) tissue classifications, (b) spatial registrations to the IXI555 Dartel template, (c) central surfaces with cortical thickness, (d) spacial registrations to the surface template, (e) RBM measurements, and finally (f) quality assessments.

  • cat12_SBM_201_extractAddSurfPara

    Call of the GUI function Extract Additional Surface Parameters to extract further surface measures such as gyrification or fractal dimension.

  • cat12_SBM_203_MapVol2IndSurf

    Map volume data from the individual volume to the individual surface.

  • cat12_SBM_204_ResampAndSmooth

    This script calls the Resample and smooth GUI function to prepare the surface data for statistical processing.

  • cat12_SBM_301_Surf2ROI

    Map single SBM measures to RBM. Extract ROI-based Surface Values

  • cat12_SBM_302_exportRBM

    Create a CSV-file (a table with ROIs columns and subject rows) for each voxel/surface-based RBM measures saved in the XML-files of multiple subjects.

  • cat12_SBM_A202_MapNormVol2AvgSurf

    Map volume data to the IXI555 Dartel surface template, e.g. to show VBM results on the surface.
