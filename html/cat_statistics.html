Statistics about CAT12 use


CAT12 uses the anonymized IP address (e.g. 192.168.100.xxx) for geolocation and sends information about Matlab version, CAT12 version, computer platform, and error messages to the SBM server, This helps us to get statistics about the use of CAT12 and more importantly about possible errors.
If you don't want to have a unique web analytics cookie identification number assigned to your computer to avoid the aggregation and analysis of data collected on this website, you can select the opt-out option at the bottom of this page.
Finally, you can set the flag "cat.extopts.send_info" in cat_defaults.m to "0" to prevent sending any anonymized information to the SBM server.


CAT12 statistics:

  • CAT12.X: Information about CAT12 version, computer platform and successful or unsuccessful preprocessings. This data is sent to the SBM server during preprocessing to obtain information about possible errors and the overall number of successful preprocessings (see pageviews).
  • Start: Information about Matlab version. This data is sent to the SBM server when CAT12 is started and provides the approximate number of (unique) users (see unique pageviews).
