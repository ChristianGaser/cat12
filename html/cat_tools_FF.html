Additional Functions

CAT12 contains some internal functions that are maybe helpful for special data processing.

Im-/export functions

  • XML im-/export

    CAT12 uses the cat_*.xml files to store additional information such as processing parameters, quality and subject measurements for each subject. You can use the cat_io_xml routine to read this data into a MATLAB structure and to extract special values of interest.
    See help cat_io_xml for further options.
    This function use xml read and write functions written by Jaroslaw Tuszynski.

  • CSV im-/export

    CAT12 can read and write csv-files for in- and export of cell-tables with values and strings.
    See help cat_io_csv for further options.

  • File search

    Linux/UNIX-like find function written by Jochen Weber (Version 0.9d, 2014).
    See help cat_vol_findfiles for further options.

Image functions

  • Statistical functions

    Replacements of MATLAB Statistical Toolbox functions to ignore NaN values:
    cat_stat_nanmean, cat_stat_nanmedian, cat_stat_nansum, cat_stat_nanstd
