ROI Tools

CAT ROI processing extracts volume/surface data for specific atlas maps and writes the results to CSV and XML-Files (see CAT Additional Tools for readout functions). Volume-based ROI data extraction is part of the CAT preprocessing pipeline were different volume-based atlas maps can be used to estimate volume, intensity, and thickness information.

  • Extract ROI-based surface values

    This function extracts ROI-based surface values by specific atlas maps.

  • Estimate Mean Values Inside ROI for External Analysis

    This function combines XML files of several subjects and saves a CSF file for further analysis using external software (e.g. Excel, SPSS).

  • Analyze ROIs

    The XML files of several subjects can be analyzed using an already existing SPM design. Here, the SPM.mat file is used in order to get information about all respective label files, but also about your design (including all covariates/confounds that you have modeled). Thus, the same statistical analysis that is saved in the SPM.mat file is applied to your ROI data.
