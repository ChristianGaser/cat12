ROI Tools (in development)

CAT ROI processing extract volume/surface data for specific atlas maps and write the results to CSV and XML-Files (see CAT further Tools for readout functions). Volume-based ROI data extraction is part of the CAT preprocessing pipeline were different volume-based atlas maps can be used to estimate volume, intensity, and thickness information.
