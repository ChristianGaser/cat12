Additional Functions

VBM12 contains some internal functions that are maybe helpful for special data processing.

Im-/export functions

  • XML im-/export

    VBM12 uses the vbm_*.xml files to store additional information such as processing parameters, quality and subject measurements for each subject. You can use the vbm_io_xml routine to read this data into a MATLAB structure and to extract special values of interest.
    See help vbm_io_xml for further options.
    This function use xml read and write functions written by Jaroslaw Tuszynski.

  • CSV im-/export

    VBM12 can read and write csv-files for in- and export of cell-tables with values and strings.
    See help vbm_io_csv for further options.

  • File search

    Linux/UNIX-like find function written by Jochen Weber (Version 0.9d, 2014).
    See help vbm_findfiles for further options.

Image functions

  • NLM Filter

    The local adaptive Non Local Means (sanNLM) works also very well on other image modalities.
    You can use it by calling the vbm_vol_sanlm function.
    See help vbm_vol_sanlm for further options.

  • Statistical functions

    Replacements of MATLAB Statistical Toolbox functions to ignore NaN values:
    vbm_stat_nanmean, vbm_stat_nanmedian, vbm_stat_nansum, vbm_stat_nanstd

  • Morphological operations

    Morphological operations like erosion, dilation, closing and opening of a binary image are available with the vbm_vol_morph function.
    See help vbm_vol_morph for further informations.

  • Kappa and RMS error estimation for test and ground truth images

    The function vbm_tst_calc_kappa allows the estimation of the Kappa measure and of the RMS error for a set of images with one (e.g. brain mask) or three classes (segment image Yp0). The first input is given by a set of test images, whereas the second input has to be one ground truth image for all or for each test image. The output is also saved as *.csv file.
    See help vbm_tst_calc_kappa for further informations.
