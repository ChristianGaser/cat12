Atlas creation and ROI based analysis (in development)

VBM12 allows estimation of tissue volume [, intensity, and thickness] for different atlas maps.

Results were stored in a csv-file vbmROI[SPACE]_[ATLAS]_*.csv for each atlas map ATLAS and in the vbm_*.XML file. SPACE describes the space of the value estimation an can be s for subject space and w for normalized space. You can use external software like Excel for analysis or the vbm_io_csv function to read the data as table, or the vbm_io_xml.

We prepared different atlas maps for the VBM12 IXI555 space that are stored in the templates_1.50mm directory. For each atlas you can find a csv-file with the ROI IDs, short and full names. The txt-file contain further informations, terms of use, and the name of the original publications and websites.

List of atlas maps in VBM12

  • Mori

  • IBSR

Atlas creation with original data (Hammers, IBSR, Mori)

If original T1 maps and subject label maps were available we used VBM12 to preprocess the data and map them to common space. The subject dependent labels were translated to the VBM12 name system.
